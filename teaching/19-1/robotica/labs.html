Laboratorio

Las sesiones de laboratorio utilizarán Linux y Python.
  • Python: Una buena introducción a Python se da en este tutorial (de Stanford CS231n). La parte correspondiente a Numpy es muy importante para este curso. Si es la primera vez que se usa Python, una introducción se puede encontrar en Codecademy.
  • Linux: Un tutorial rápido de líneas de comando se puede encontrar aquí. Se debe leer al menos el primer y segundo tutorial.
El código utilizado en los laboratorios se encuentra aquí (repositorio de github).
